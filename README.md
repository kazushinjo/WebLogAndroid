# WebLog for Android

アマチュア無線の交信記録（QSO ログ）を管理する Android アプリです。
Jetpack Compose で実装されており、スマートフォン・タブレット両方に対応します。

## 主な機能

- QSO（交信）の記録・編集
- マイコールサイン管理
- コールサイン入力時の重複チェック
- QTH 入力時の JCC（市区町村）サジェスト（1,768件収録）
- 周波数入力によるバンド自動判別
- バンド・モードによる絞り込みフィルタ
- ADIF 形式でのエクスポート
- バックアップ JSON の保存・読み込み
- タブレット向け左右2画面レイアウト
- 入力欄以外をタップするとキーボードを自動で閉じる

## 動作環境

| 項目 | 要件 |
|------|------|
| Android | 8.0 (API 26) 以上 |
| 確認済み端末 | I11_Power (Android 13) |
